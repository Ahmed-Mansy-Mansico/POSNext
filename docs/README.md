### For Developers

**Check current version:**
```bash
cd /home/ubuntu/frappe-bench
bench --site nexus.local execute pos_next.utils.get_app_version
```

**Bump version:**
```bash
cd /home/ubuntu/frappe-bench/apps/pos_next
./scripts/version-bump.sh patch  # or minor/major
```

**Build frontend:**
```bash
cd POS
yarn build
```
